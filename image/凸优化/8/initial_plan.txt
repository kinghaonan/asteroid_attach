        * Switching logic: Use spherical harmonics outside Brillouin sphere, Bessel inside
    - Parameters: C_lm, S_lm (degree 4), Ā_l,n,m, B̄_l,n,m (l_max=2, n_max=5, m_max=5)
    - Dependencies: scipy.special for Legendre and Bessel functions

    COMPONENT 2: NONLINEAR PROBLEM FORMULATION (P1)
    - Purpose: Define original nonconvex optimal control problem
    - Files: optimization/problem_formulation.py
    - Equations (Eqs.8-16):
        * State: x = [r, v, m] where r position, v velocity, m mass
        * Dynamics: ṙ = v, v̇ = T/m + g(r) - 2ω×v - ω×(ω×r) - ω̇×r
        * Mass flow: ṁ = -∥T∥/(I_sp * g_0)
        * Constraints: T_min ≤ ∥T∥ ≤ T_max, m ≥ m_dry, glide slope (Eq.13)
        * Objective: minimize J = -m(t_f) (maximize final mass)
    - Variables: Continuous time t ∈ [0, t_f]

    COMPONENT 3: LOSSLESS CONVEXIFICATION ENGINE
    - Purpose: Transform P1 into convex SOCP problem P3
    - Files: optimization/problem_formulation.py
    - Transformations (Section III):
        * Slack variable: Introduce T_m with ∥T∥ ≤ T_m, T_min ≤ T_m ≤ T_max
        * Variable change: a_t = T/m, a_tm = T_m/m, q = ln(m)
        * Taylor expansion about q₀ = ln(m₀):
          T_min e^{-q₀}[1 - (q-q₀) + 0.5(q-q₀)²] ≤ a_tm ≤ T_max e^{-q₀}[1 - (q-q₀)]
        * Final SOCP constraint: ∥a_t∥ ≤ a_tm (second-order cone)
    - Validation: Must check ∥a_t∥ ≈ a_tm (error < 1e-4%) at solution

    COMPONENT 4: SUCCESSIVE SOLUTION SOLVER
    - Purpose: Iteratively solve sequence of SOCPs to handle nonlinear gravity
    - Files: optimization/successive_solver.py
    - Algorithm (Section IV):
        1. Initialize reference trajectory r_ref(t) = initial guess
        2. While max∥r_new(t) - r_ref(t)∥ > 0.5m:
           a. Linearize gravity: g(r) ≈ A(r_ref)r + c(r_ref)
              where A = dominant term (-μ/r³ for spherical, Eq.57 for Bessel)
           b. Form SOCP with linearized dynamics (Eq.56)
           c. Solve SOCP to get new trajectory r_new(t)
           d. Update r_ref = r_new
        3. Return converged trajectory
    - Parameters: Convergence tolerance 0.5m, max iterations 10
    - Discretization: Trapezoidal rule with Δt = 2s (Eq.58)

    COMPONENT 5: FLIGHT TIME OPTIMIZER (BRENT'S METHOD)
    - Purpose: Find optimal flight time minimizing propellant
    - Files: optimization/flight_time_optimizer.py
    - Algorithm (Section V):
        1. Define function f(t_f) = propellant_used(t_f) via inner SOCP solver
        2. Use Brent's method on f(t_f) over [t_min, t_max]
        3. Coarse search: Δt = 10s for initial Brent iterations
        4. Fine solution: Δt = 2s at optimal t_f*
    - Properties: f(t_f) is unimodal (Figs.5-7)
    - Expected evaluations: 7 for full thrust, 9-11 for quarter thrust

    COMPONENT 6: CONSTRAINT SYSTEM
    - Purpose: Implement trajectory constraints
    - Files: optimization/constraints.py
    - Constraints:
        * Glide slope: ∥r - r_f∥ cosθ - (r - r_f)·n̂ ≤ 0 (Eq.13,62)
        * Vertical motion: (r - r_f)·n̂ ≥ 0 for final phase (Eqs.60-61)
        * Mass limit: q ≥ ln(m_dry)
        * Boundary conditions: r(0)=r₀, v(0)=v₀, m(0)=m_wet; r(t_f)=r_f, v(t_f)=v_f

    COMPONENT 7: SCALING SYSTEM
    - Purpose: Numerical conditioning for SOCP solver
    - Files: dynamics/scaling.py
    - Scaling factors (Eq.59):
        * Distance: R_sc = γ (smallest semi-major axis)
        * Velocity: v_sc = √(R_sc * g_sc)
        * Acceleration: g_sc = μ / R_sc²
        * Time: t_sc = √(R_sc / g_sc)
        * Mass: m_sc = 1 (use q = ln(m))
    - Implementation: Scale all variables before SOCP, unscale after
