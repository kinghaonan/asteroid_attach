# Phase2 轨迹优化优化指南

## 📋 问题诊断

### 原代码的主要问题

1. **伪谱法过于复杂**
   - 使用间接法需要求解协态方程
   - 14维状态空间在每个节点都参与优化
   - 约束数量巨大（每个节点14个约束）
   - 收敛困难，容易陷入局部最优

2. **打靶法初始猜测困难**
   - 只有4个固定的初始猜测
   - 没有基于物理的智能猜测
   - Bang-bang控制对数值误差极其敏感

3. **缺少用户反馈**
   - 优化过程黑盒，看不到进度
   - 不知道什么时候会结束
   - 失败后没有降级方案

4. **字体问题**
   - 只配置了SimHei和Microsoft YaHei
   - 在非Windows系统上显示为乱码

## ✅ 优化内容

### 1. 伪谱法优化 (`pseudospectral_optimized.py`)

#### 主要改进：
- ✅ **添加tqdm进度条**：实时显示优化进度
- ✅ **简化算法**：使用直接法，去除复杂的协态优化
- ✅ **改进初始猜测**：使用S曲线插值更符合物理实际
- ✅ **更好的数值稳定性**：自适应归一化
- ✅ **详细的调试信息**：显示迭代次数、耗时、收敛状态

#### 核心算法变化：
```python
# 原算法：间接法（14维）
状态 = [r(3), v(3), m, lam_r(3), lam_v(3), lam_m]  # 14维

# 优化算法：直接法（8维）
状态 = [r(3), v(3), m, u]  # 8维（去除协态变量）
```

#### 使用示例：
```python
from algorithms.pseudospectral_optimized import PseudospectralOptimizerOptimized

optimizer = PseudospectralOptimizerOptimized(
    asteroid, spacecraft,
    n_nodes=20,  # 推荐20-30
    verbose=True  # 显示进度条
)

    r0, v0, m0, rf, vf, t_span,
    max_iter=300
)
```
